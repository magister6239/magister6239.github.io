Нажмите кнопку Слайды выше, чтобы просмотреть встроенную функцию слайдов.

Слайды можно добавлять несколькими способами:

  • Создавайте слайды с помощью функции Slides Hugo Blox Builder и связывайте их с помощью параметра slides в начале файла обсуждения.
  • Загрузить существующую подборку слайдов в static/ и связать ее с помощью параметра url_slides в начале файла обсуждения.
  • Встройте свои слайды (например, Google Slides) или видеопрезентацию на эту страницу с помощью коротких кодов.

В тело этой страницы можно добавить дополнительную информацию о событии, включая элементы страницы, например галереи изображений.
